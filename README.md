# Blue Drift 3D

깊은 물속에서 푸른 해파리가 천천히 숨을 쉬는 인터랙티브 물멍 시뮬레이션입니다.

## 이번 업로드 버전

- GitHub Pages 업로드용 정적 파일
- 업로드한 MP3 파일을 `music.mp3`로 적용
- Web Audio buffer loop 방식으로 음악 반복 재생
- 해파리 움직임, 부유물, 촉수 반응은 최종 버전 유지

## 업로드할 파일

GitHub 저장소 루트에 아래 파일들을 그대로 올리세요.

```text
.nojekyll
index.html
main.js
music.mp3
README.md
style.css
```

## GitHub Pages 설정

1. 저장소의 Settings로 이동
2. Pages 메뉴 선택
3. Source를 `Deploy from a branch`로 선택
4. Branch를 `main`, 폴더를 `/root`로 선택
5. Save
6. 배포 주소가 생성될 때까지 1~3분 정도 기다리기

## 주의

zip 파일 자체를 올리는 것이 아니라, 압축을 풀고 안의 파일들을 업로드해야 합니다.
