# Blue Drift 3D 5174 Version

localhost:5174 때의 촉수 반응 느낌을 기준으로 정리한 GitHub Pages 업로드용 정적 파일입니다.

## 업로드 방법

이 폴더 안의 파일들을 GitHub 저장소 루트에 그대로 올리세요.

필수 파일:
- index.html
- style.css
- main.js
- music.mp3
- .nojekyll
- README.md

## GitHub Pages 설정

1. GitHub 저장소로 이동
2. Settings → Pages
3. Source를 `Deploy from a branch`로 선택
4. Branch를 `main`, 폴더를 `/root`로 선택
5. Save
6. 배포 주소가 뜰 때까지 1~3분 정도 기다리기

주의: zip 파일 자체를 올리는 것이 아니라, 압축을 풀고 안의 파일들을 올려야 합니다.
