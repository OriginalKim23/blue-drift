# Blue Drift 3D

깊은 물속에서 푸른 해파리가 천천히 숨을 쉰다.  
빛, 촉수, 부유물, 소리가 함께 반응하는 인터랙티브 물멍 시뮬레이션.

## 소개

**Blue Drift 3D**는 하나의 푸른 해파리를 중심으로 만든 3D 인터랙티브 웹 경험입니다.

사용자는 화면을 드래그해 시야를 천천히 돌릴 수 있고,  
해파리의 갓을 터치하면 빛이 퍼지며 촉수가 물결처럼 반응합니다.  
촉수를 문지르면 가닥마다 은은하게 빛나고, 위치에 따라 다른 소리가 납니다.

게임보다는 관찰에 가깝고,  
조작보다는 천천히 바라보는 경험을 목표로 합니다.

## 주요 기능

- 3D 해파리 렌더링
- 깊은 바닷속 느낌의 어두운 공간
- 흰 부유물과 먼지 입자를 통한 공간감
- 갓 터치 시 발광 반응
- 갓 터치에 따른 촉수 웨이브 반응
- 촉수 문지르기 시 가닥별 빛 반응
- 촉수 위치에 따른 음 높낮이 변화
- 드래그를 통한 시야 회전
- 휠 또는 핀치로 접근/후퇴
- 수중음, 음악, 터치음 볼륨 조절

## 조작 방법

| 조작 | 기능 |
|---|---|
| 드래그 | 시야 회전 |
| 마우스 휠 / 핀치 | 접근 또는 후퇴 |
| 해파리 갓 터치 | 발광 및 촉수 웨이브 |
| 촉수 문지르기 | 가닥별 빛 반응과 소리 |
| 설정 버튼 | 수중음, 음악, 터치음 조절 |
