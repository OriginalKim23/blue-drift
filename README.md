# Blue Drift Static Easy Upload

이 버전은 GitHub Actions, npm, Vite 없이 GitHub Pages에서 바로 열 수 있는 정적 버전입니다.

## GitHub에 올릴 파일
아래 파일만 저장소 최상단에 올리면 됩니다.

- index.html
- main.js
- style.css
- music.mp3
- README.md

## GitHub Pages 설정
Settings → Pages

- Source: Deploy from a branch
- Branch: main
- Folder: /root

Save 후 1~3분 기다리면 됩니다.
