## 이번 수정
- 첫 화면 문구 교체
- 버튼 클릭 후 2~3초 안에 검은 화면에서 해파리가 서서히 드러나는 연출 추가
- 갓 터치 반응을 생물체의 젤리 같은 펌핑으로 조정
- 빛 반응을 순간 반짝임보다 부드러운 차오름/사라짐으로 조정
- 갓 터치 시 번진 푸른 링 잔광 추가
- 촉수 문지름 시 터치한 주변 가닥마다 작은 빛 조각이 생기도록 추가
- 촉수 소리는 몽환적인 기계/글라스 신호음 유지
